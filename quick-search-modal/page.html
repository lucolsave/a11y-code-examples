Key Implementation Notes

Search Input

These attributes are recommended for the input field inside the dialog:
  • The input element should use type="search".
  • aria-controls: Should reference the ID of the element containing search suggestions. (Note: This element should always exist in the DOM, even if it is hidden initially.)
  • autocomplete="off": Turns off automated assistance in filling out form field values.

Also note

For this example we didn't to add a tab trap inside dialog, though this is recommended. If you make the dialog a modal (traps navigation), add the aria-modal attribute to your dialog element
