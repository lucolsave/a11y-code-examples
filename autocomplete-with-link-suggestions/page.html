    Key Implementation Notes

    • Suggestions don't use a native <a> as it's link since elements inside role="option" loose their semantic value. Also, list boxes aren't meant to display links. This workaround adds a click event to each link with javascript.
    • Each option has it's function described by via the aria-describedby attribute. This could be considered bad practice as it's repetitive and it also doesn't utilizes screen readers build in abillity to identifiy links.
    • The idea is that the user should be redirected to a search page with the selected query once a suggestion is clicked. Topics and recepies takes the user directly to the page.
